Possibly outdated

These AWS EC2 tips and howtos was written mostly by me in 2009 when AWS was a only just out of public beta.
Since then AWS has exploded into a huge ecosphere of services and tools.

I consider these tips as still useful as an introductory and general advice, but some may be a bit outdated or no longer relevant.

I use Amazon's ec2 for most of my server needs and then some.
And have decided to document some of these uses, for others benefit.

This is relevant if you:

  • Are interested in running servers in the cloud...
  • Have a preference for Ubuntu...
  • Need tips on how to use these servers...
  • Need step by steps how to set up apache, tomcat, postfix etc...
