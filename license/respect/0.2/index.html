Introduction

flurdy's license of respect is a license to clarify some ethical suggestion on top of the Creative Commons License that is part of it. These are optional elements, but should morally be applied.

People who adhere to these suggestions will be celebrated.
People whom ignore them, will be shunned.

You may refer to anything licensed with flurdy's respect license as a "Creative Commons by-sa" license, if you prefer.

Creative Commons

At the core of this license is the Creative Commons by-sa license, which is their Attribution-ShareAlike: 2.5 license. Here is the human readable summary: creativecommons.org/licenses/by-sa/2.5/.

This means all uses of this work has to be properly attributed to the owner. It also means people can share/modify/distribute the work, as long as it is also licensed with the same license. Your work using this work may also be used commercially.

Gnu Lesser Public License

This license is also quite closely linked to the Gnu's Lesser Public License, LGLP. Which is the option often chosen for sourceforge projects, due to its lack of Creative Commons copyright licenses. However where available it is prefered to use Creative Commons as it is less militant and pedantic.

Suggestions

These are this license's ethical suggestions on top of the requirements of the Creative Commons license. What we aim to achieve is to improve the work by: spreading the word so that more people use and work on it, share these improvments, and rewarding people for working on it.

  • Attribution

    • Minimum

      These are suggested minimum way of attributing your work's usage of our work. If you only rely on it slighlty or just started using it, it is ethically fine to do this.

      Examples:
      • Web site or web application

        Include a paragraph in the about page. With relevant links to this license page and the work covered by this license's page.

      • Downloadable package

        Include a paragraph in the release notes. With relevant links to this license page and the work covered by this license's page.

    • Reliance

      If your work is is heavely reliant of the work covered by this license, then this license suggest you should attribute more than the minimum.

      Examples:
      • Web site or web application

        A text sentence, tag image or logo link to the project on the front page. More than a paragraph in the about page, perhaps a seperate document about the work and its association with your work.

      • Downloadable package

        More than a paragraph in the release notes, perhaps a seperate document about the work and its association with your work. The description of the project it self should perhaps mention this work.

  • Share

    The work associated with this license: E.g. text,code,images,audio etc.

    • Modifications

      If you want to or have made modifications, tweaks, enhancements, alternatives etc of this work, then please share this with us in a proper way. Submit patches, comment in forums, add tasks, add alternative modules etc so that others are aware of these modidications. Hiding it in the deepest darkest corners of your web site is not the easiest way for us to find out about it.

    • Extensions

      If you extend this work, or integrate this with other work. Then please share this work with the same license. All or parts of it, or just some of the knowledge you gained from it.

  • Donations

    While not required it would be respectfull if you donated some back to the people that helped create this work. Especially if you are a commercial organisation.

    • Usage

      If you are using this work a lot, depend on it even, then please contribute by makeing a one off donation to the people/organisation/project behind this work. This should reflect the use and size of your organisation.

      Examples:
      • Single person, slight usage: £/$/€0
      • Single person, heavy usage: £/$/€25
      • Small organisation, slight usage: £/$/€50
      • Small organisation, heavy usage: £/$/€1000
      • Large organisation, slight usage: £/$/€500
      • Large organisation, heavy usage: £/$/€10000
    • Reveue

      If you or your organisation is making revenue by using this work, then please attribute some of this back to the creators of this work. If your product is selling well, and this work is a part of it but maybe not a core part, then still make a donation. If you are very productive and this work is a core part of the reason for your revenue turn over, then please make a sizable donation. Please make a recuring donation to reflect the continuing use of this work.

      Examples:
      • Slight revenue, non essential: £/$/€5 now and then
      • Large revenue, non essential: £/$/€50 now and then
      • Slight revenue, core: £/$/€25 frequently
      • Large revenue, core: £/$/€1000 frequently
